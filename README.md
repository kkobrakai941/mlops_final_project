# Проект MLOps: Система прогнозирования оттока клиентов

## Цель проекта
Целью проекта является создание энд ту энд MLOps системы, которая решает бизнес задачу прогнозирования оттока клиентов. Сервис принимает данные о пользователях и предсказывает вероятность того, что клиент прекратит пользоваться услугой. Это позволяет бизнесу своевременно принимать меры по удержанию клиентов и повышает доход.

## Описание решения
Система включает несколько компонентов:
- **ML сервис** на FastAPI, который выполняет инференс модели. Модель обучается на исторических данных о клиентах и их поведении.
- **База данных PostgreSQL** для хранения исходных данных, результатов предсказаний и метаданных.
- **Брокер сообщений Kafka** для передачи данных в режиме реального времени между компонентами.
- **Планировщик Airflow** для периодического обновления витрин, расчёта фич и оффлайн предсказаний.
- **Веб интерфейс Streamlit**, позволяющий отправлять данные для предсказания и просматривать историю прогнозов и ключевые метрики.
- **Мониторинг Grafana + Prometheus** для отслеживания технических и бизнес метрик в реальном времени.

## Как запускать
Полная система собирается и разворачивается с помощью `docker compose` одной командой. Файл `docker‑compose.yml` содержит определения всех сервисов, их переменных окружения и связи. В разделе «Документация» описаны шаги по запуску, настройке окружения и взаимодействию с сервисом.
